# Methodology

## Research Approach

This project follows a **"follow the money"** methodology — tracing the intersection
of federal enforcement authority, state constitutional frameworks, and municipal
regulatory power to identify viable pathways for local election protection ordinances.

## Data Sources

All analysis on this site draws from publicly available primary sources:

| Source | Type | Data Used |
|--------|------|-----------|
| 18 USC 592 | Federal statute | Prohibition on troops at polls |
| 18 USC 1385 | Federal statute | Posse Comitatus Act |
| State constitutions | Primary law | Home rule provisions |
| Municipal code databases | Local law | Existing election-related ordinances |
| FEC enforcement records | Federal agency | Historical enforcement patterns |
| State lobbying statutes | State law | Registration requirements, thresholds, exemptions |

## Analytical Framework

### State Viability Assessment

Each state receives a composite viability rating based on:

1. **Home rule strength** — Constitutional vs. legislative home rule; breadth of municipal authority
2. **Dillon's Rule constraints** — Whether the state follows Dillon's Rule and how strictly
3. **Political feasibility** — Current political landscape, existing election protection measures
4. **Lobbying registration burden** — Cost, complexity, and timeline for compliance
5. **Litigation environment** — State court receptivity to election protection claims

### Tiered Classification

- **Tier 1 (High Viability)**: Strong home rule, favorable politics, manageable lobbying compliance
- **Tier 2 (Moderate Viability)**: Mixed factors requiring strategic approach
- **Tier 3 (Low Viability)**: Dillon's Rule constraints, hostile political environment, or prohibitive compliance costs

## Citation Standards

Every data point on this site includes:

1. **Primary source link** — direct link to the original statute, constitution, or official publication
2. **Date of access** — when the source was last verified
3. **Relevant excerpt** — the specific provision or data point cited
