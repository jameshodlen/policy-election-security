# State Authority

## Home Rule vs. Dillon's Rule

The viability of municipal election protection ordinances depends fundamentally on whether
a state grants its cities **home rule authority** or constrains them under **Dillon's Rule**.

### Home Rule States

Home rule grants municipalities inherent authority to govern local affairs without
specific state legislative authorization. Cities in strong home rule states can adopt
ordinances on any matter of local concern not preempted by state or federal law.

!!! tip "Key for Campaign Viability"
    States with **constitutional home rule** (as opposed to merely legislative home rule)
    provide the strongest foundation for municipal election protection ordinances,
    because the authority cannot be easily revoked by the state legislature.

### Dillon's Rule States

Under Dillon's Rule, municipalities possess only those powers:

1. **Expressly granted** by the state legislature
2. **Necessarily implied** from those express grants
3. **Essential** to the municipality's declared purposes

Any doubt about whether a power exists is resolved **against** the municipality.

### Mixed States

Many states apply a hybrid framework:

- Home rule for charter cities, Dillon's Rule for general law cities
- Home rule for some subjects, Dillon's Rule for others
- Constitutional home rule with significant statutory preemption

## Implications for the Campaign

| Category | Approach | Example States |
|----------|----------|----------------|
| Strong home rule | Direct ordinance adoption | IL, CO, OR, OH |
| Legislative home rule | May need state enabling legislation | MD, MA, RI |
| Dillon's Rule | Requires state authorization first | VA, NH, ID |
| Mixed | Target charter/home rule cities only | WA, MN, PA |

## Preemption Analysis

Even in home rule states, municipalities face potential preemption challenges:

1. **Express preemption** — State law explicitly prohibits local election regulation
2. **Implied preemption** — State election code so comprehensive it occupies the field
3. **Conflict preemption** — Local ordinance conflicts with specific state provisions

The campaign must analyze preemption risks state by state. See individual [State Guides](../state-guides/index.md) for jurisdiction-specific preemption analysis.
