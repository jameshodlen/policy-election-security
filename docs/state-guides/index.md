# State Guides

## 50-State Viability Assessment

Each state guide provides a comprehensive analysis of municipal ordinance viability,
including home rule authority, preemption risks, lobbying registration requirements,
and litigation environment.

### Tier 1 — Highest Viability

Strong home rule, favorable political environment, manageable compliance.

| State | Home Rule | Key Factor |
|-------|-----------|------------|
| Illinois | Constitutional | Broad home rule, strong municipal authority |
| Colorado | Constitutional | Home rule tradition, election protection history |
| Massachusetts | Constitutional (limited) | Strong local governance tradition |
| Maryland | Legislative + charter | Charter county/city authority |
| Oregon | Constitutional | Initiative/referendum culture |
| Rhode Island | Constitutional | Compact jurisdiction, accessible legislature |

### Tier 2 — Moderate Viability

Mixed factors requiring strategic targeting of specific cities.

### Tier 3 — Near-Insurmountable

Dillon's Rule constraints, hostile political environment, or prohibitive compliance.

| State | Key Barrier |
|-------|-------------|
| Idaho | Strict Dillon's Rule, preemption risk |
| South Dakota | Limited home rule, political environment |
| Wyoming | Small population, Dillon's Rule |
