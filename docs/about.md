# About

## About This Project

Election Security Research is a policy research platform focused on municipal enforcement
of 18 USC 592 — the federal statute prohibiting the use of troops at polling places.

This project supports a nationwide municipal ordinance campaign targeting approximately
180 cities, providing legal analysis, lobbying compliance guidance, and litigation
strategy for state court civil actions.

This project is part of a network of independent policy research platforms
dedicated to rigorous, primary-sourced analysis of public policy.
