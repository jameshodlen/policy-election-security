# Election Security

Municipal ordinance toolkit and 50-state analysis for polling place protection under 18 U.S.C. § 592

Part of the [Policy Research Platform](https://github.com/jameshodlen) — primary-sourced public data tracing money flows through public systems to reveal structural contradictions.

## Methodology

Follow the money. Every analysis uses publicly available primary sources, automated data pipelines, state-level comparisons anchored in Wisconsin and Minnesota, and transparent methodology.

## Structure

- `data/` — Raw and processed datasets from public sources
- `pipelines/` — Automated data ingestion and transformation scripts
- `analysis/` — Research outputs, calculations, and findings
- `site/` — Web content for the public-facing research site
- `docs/` — Methodology documentation and research notes
